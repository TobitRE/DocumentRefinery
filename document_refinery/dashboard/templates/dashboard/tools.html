Dashboard tools

{% endblock %} {% block content %}
Document upload Upload PDFs, DOCX, PPTX, or XLSX files with structured Docling controls and JSON fallback. Profile comparison Queue comparison jobs for an existing document across backend-defined Docling profiles. Runtime diagnostics Inspect Docling packages, runtime settings, broker/worker state, filesystem paths, OCR, and FFmpeg. Job inspection Filter ingestion jobs and open job details with artifacts, options, metrics, and retry/cancel controls. API keys Create or update tenant keys, scopes, upload MIME limits, and default Docling options. Webhook endpoints Review delivery targets and connect job update events to tenant-specific endpoints.

Boundaries

Implemented
PDF, DOCX, PPTX, and XLSX upload, PDF profile presets, Docling JSON/markdown/text/doctags exports, figure zip metadata, runtime diagnostics, job and webhook inspection.
Planned later
Real chunking and VLM inference remain future work until backend support is explicit.
Not offered here
Remote Docling services, arbitrary plugin execution, and unrestricted file ingestion are intentionally outside this dashboard.
